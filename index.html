CPE232 this course focuses on managing enterprise servers with a DevOps culture. In this course, Discover how to use Ansible for configuration management and de facto standard monitoring procedures to monitor and utilize alerts in a DevOps environment. You will also master the fundamentals of operating a DevOps environment on Linux servers. Ubuntu

A Linux distribution made primarily of open-source and free software, with a Debian foundation. Official releases of Ubuntu are available in three editions: Desktop, Server, and Core for Internet of Things gadgets and robotics.

Linux

An open source operating system (OS) is called Linux®. The software that controls a machine's hardware and resources—such as CPU, memory, and storage—is called an operating system.

Ansible

A Python-based command-line tool for IT automation that is free source. In order to facilitate application deployment, system upgrades, and other activities, it may setup systems, distribute software, and coordinate sophisticated processes.

Scripting

Utilized mostly to leverage an existing software while automating chores for websites and online apps. Information extraction from a dataset is one of its uses.

Github

Github is a site that hosts code for version control and teamwork. It enables remote collaboration on projects between you and other people. You will learn the fundamentals of GitHub, including repositories, branches, commits, and pull requests, in this lesson.

CentOS

Built on the Linux kernel and derived from the Red Hat Enterprise Linux source code, CentOS is an open-source operating system.
