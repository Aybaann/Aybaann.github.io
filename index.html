Learnings

CPE232 this course focuses on managing enterprise servers with a DevOps culture. In this course, Discover how to use Ansible for configuration management and de facto standard monitoring procedures to monitor and utilize alerts in a DevOps environment. You will also master the fundamentals of operating a DevOps environment on Linux servers
Taking Managing Enterprise opened up more pathways for me, especially to the word Linux wherein it is a free open source operating system where we can practice our programming skills. I learned a lot this semester in using Ansible, roles, monitoring tools, and GitHub. My first encounter with Ansible made my life miserable since it had a lot of error, especially in installing and using it, but with time, I enjoyed doing Ansible since it makes the developer works faster than a normal installation process. Learning roles makes my workplace clean and neat, and creating roles helps me find my errors, especially in the installation process. Monitoring tools, I'm still learning their uses in the programming, but understanding its concepts allows the user to track progress and makes the user's life easy in doing the project. Lastly, Github is a lifesaver for us since it saves our progress on its website, and we can do our working progress in no time..
